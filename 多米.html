公司定位与优势

公司定位

本公司专注弱电智能化建设与整体方案服务，按客户预算、场景和目标做多品牌组合。

  • 前期：现场勘查、需求分析、方案设计。
  • 中期：标准化施工、设备安装、系统调试。
  • 后期：快速响应售后、巡检维护、培训交付。

经营与代理优势

  • iKuai 企业网关、交换机、无线路由、无线 AP 方案齐全。
  • Dahua / 海康适合监控、录像机、周边 POE 交换机组合。
  • MikroTik 适合复杂分流、策略路由、多 WAN、VPN。
  • 华为适合核心、汇聚、接入、防火墙、AC / AP 体系化部署。
  • 锐捷 / TP-Link / H3C / UniFi / Ruckus 可按项目层级灵活组合。
培训重点：员工介绍公司时，不要只说“卖设备”，要讲“我们做的是一站式弱电智能化方案”。

客户沟通与成交话术

初次接触

  • 先问客户是新建项目还是升级改造。
  • 再问主要解决网络、监控，还是整体弱电问题。
  • 最后讲公司优势：方案稳定、组合灵活、后续维护方便。

客户必问问题

  • 项目地点、面积、房间数量、终端数量。
  • 宽带数量、带宽大小、是否跨境访问。
  • 监控点位、存储天数、夜晚要求。
  • 是否需要高密 Wi‑Fi、漫游、远程管理。

典型项目场景

中小型企业（50-100人）

  • 稳定 Wi‑Fi 覆盖，基本行为管理。
  • 前台与出入口监控，员工考勤门禁。
  • 推荐：iKuai / TP / 锐捷 + 海康 / 大华。

连锁门店

  • 收银系统稳定，断电保护。
  • 无死角监控，手机远程看店。
  • 推荐：基础网关 + AP + 监控 + NVR。

酒店 / 民宿

  • 客房面板 AP 美观，走廊到房间不断网。
  • 大堂、走廊、停车场高清录像。
  • 推荐：面板 AP + 星光 / 夜视型监控。

园区 / 工厂 / 高并发场景

  • 高吞吐量核心骨干，高可用。
  • 综合大屏中心，重点区域集中管理。
  • 推荐：华为 / MikroTik / H3C / Ruckus / UniFi 组合。

新人30天成长计划

第 1-2 周

  • 认识产品外观、接口、品牌和分类。
  • 熟悉内部系统和基础流程。

第 3-4 周

  • 学习沟通话术、需求判断、方案配单。
  • 理解组网拓扑，开始模拟跟单。

新人培训重点

先分类型

  • 摄像机 / 录像机 / 交换机 / 路由器 / AP / 控制器先分清。
  • 不要一上来先背型号。
  • 先知道每一类设备是干什么的。

先问需求

  • 监控先问安装位置、夜晚效果、距离、存储。
  • 网络先问人数、POE、VLAN、远程管理。
  • 无线先问房间型还是大厅型、并发量、是否统一管理。

会做组合

  • 监控：摄像机 + NVR + 硬盘 + POE交换机。
  • 网络：路由 / 防火墙 + 交换机 + AP。
  • 高密无线：高规格 AP + POE交换机 + 控制器 / 网关。

产品类目说明

监控类

  • 半球摄像机：常用于室内、前台、走廊、电梯口。
  • 枪机摄像机：常用于室外、围墙、停车场、门口。
  • 球机 / PTZ：用于大范围巡航、远距离拉近、重点区域查看。
  • NVR录像机：按 8 / 16 / 32 / 64 路去理解，再看盘位数量。

网络类

  • POE交换机：适合给 AP、摄像机等设备供电。
  • 普通交换机：只做网络转发，不直接供电。
  • 三层交换机：适合核心、汇聚、VLAN 较多的项目。
  • 企业路由 / 网关：用于出口、分流、限速、行为管理、VPN。
  • AP：吸顶、面板、室外、高密，需要按场景推荐。
  • 无线控制器：适合多 AP 统一管理。
销售提醒：让员工学会按场景推荐，不是死背品牌和型号。
